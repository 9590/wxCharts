Detailed Description

This class is used to display a tooltip.

Charts can display tooltips in response to a user action (e.g. hoverin with the mouse over some part of the chart). This class implements such tooltips. Unlike the wxChartTooltip class this class can display information about multiple items at once for those case where the user selects more than one item at a time.
